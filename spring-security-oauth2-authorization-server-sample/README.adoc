这是一个 Spring Security Oauth2 Authorization 的示例项目

因为所有的存储器都是使用的内存存储, 所以直接启动即可.

1. 运行 `Application`

2. 模拟了第三方调用Oauth服务.

2.1 浏览器打开: `http://localhost:8080/third/authorization`
